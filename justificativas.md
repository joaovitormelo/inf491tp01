# Instrumentos

## Instrumento 1: Guitarra
- Literatura base: [Como sintetizar som da guitarra](https://soundcy.com/article/how-to-synthesize-guitar-sound#:~:text=Synthesizing%20guitar%20sounds%20involves%20replicating%20the%20unique%20tonal,guitar%20acoustics%20to%20create%20authentic%20or%20experimental%20tones.)
    - Recomendação do uso de múltiplas formas de onda combinadas:
        - A forma senóide atua como se fosse a corda "dry" (pensando na lógica wet-dry), sem nenhum efeito aplicado;
        - Ondas quadradas, ricas em harmônicos, adicionam som metálico
        - Ondas "dente de serra" produzem um som com "brilho", podendo simular algumas distorções da guitarra.
    - Além disso, também aplicamos AM de alta frequência para adicionar efeito "metálico" (usando AM para modificação de timbre) e FM para adicionar um leve vibrato.
    - Procuramos deixar vários harmônicos para enriquecer o som (10 pra cada forma de onda)
